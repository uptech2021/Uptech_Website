## Setup Instructions

To run the project, use the pnpm package manager. Follow these steps:

1. Install the dependencies:
   ```bash
   pnpm install
   ```

2. Start the development server:
   ```bash
   pnpm dev
   ```

Contact the lead senior developer, Aaron, for the `.env` file.
